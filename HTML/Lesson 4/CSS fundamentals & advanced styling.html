the CSS box model

evert HTML element is a rectangle with 4 components

  • contnet - the actual text or image
  • padding - a space between the content and the border
  • border - the line around the padding
  • margin - space outside the border

calculating box dimentions

when you set the width or height or an element you are defining the dimentions of the content box

padding, borders and margin are then added on top giving the total space of the element ocupied the formulas are:

Total Width = Width + Left/Right Padding + Left/Right border + Left/Right Margin

Total Height = Width + Left/Right Padding + Left/Right border + Left/Right Margin
