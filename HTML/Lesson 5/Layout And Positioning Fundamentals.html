Layout and Positioning Fundamentals

Understanding the display property

The display property is used to define how an element is displayed on the page.

Block elements - full width new line

display: block;

Inline elements - only as wide as the content

display: inline;

Inline-block elements - only as wide as the content but can have width and height

display: inline-block;

hide elements - the element will not be displayed on the page

display: none;

Block vs Inline vs Inline-block

Block Elements

  • Take the fully available width of the parent element
  • Always start on a new line
  • Can have width, height, margin, and padding
  • Examples: div, p, h1-h6, ul, ol, li, form

Inline Elements

  • Only take up as much width as needed
  • Do not start on a new line
  • Cannot have width, height, margin, or padding
  • Examples: span, a, img

Inline-Block Elements

  • Only take up as much width as needed
  • Do not start on a new line
  • Can have width, height, margin, and padding
  • Examples: button, input

Navigation styling with Display

to make a navigation bar you use nav

to style the nav bar you use display: flex; to make the list items display in a row

to make the list items display in a row you use justify-content: center;

to remove the bullet points you use list-style: none;

to add space between the list items you use gap: 2rem;

to style the links you use text-decoration: none; to remove the underline

to style the links you use color: black; to change the color of the text

to style the links you use padding: 0.5rem 1rem; to add space around the text

to style the links you use border-radius: 1px; to round the corners of the links

to style the links you use transition: background-color 0.3s ease; to make the background color change smoothly

Positioning

Positioning is used to define how an element is positioned on the page.

  • Static - the default position of an element
  • Relative - the element is positioned relative to its normal position
  • Absolute - the element is positioned relative to its nearest positioned ancestor
  • Fixed - the element is positioned relative to the browser window
  • Stacking - the element is positioned on top of other elements
